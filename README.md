🐑 羊羊大作战（QiuQiuDaZhan 改进版）
项目简介

羊羊大作战 是一个基于 Qt 的小游戏项目，灵感来源于经典球球吞噬类游戏。玩家控制自己的球球，通过吃掉小球和敌人球来成长，同时躲避比自己大的球。游戏支持皮肤选择、道具、敌人 AI、以及可视化的星空背景。

🎮 游戏玩法
控制小球：用鼠标拖动控制玩家球移动
吞噬规则：
吃掉比自己小的球和食物可以增长
被比自己大的球碰撞时，会逐渐减小并降低分数
生命值与分数：
玩家球有生命值和分数
分数降为 0 或被完全吞噬时游戏结束
道具系统：
随机生成加速、护盾、回血道具
吃到道具会触发效果
敌人球：
支持随机生成和持续出现
可以吃小球成长
敌人球皮肤随机分配
🌟 界面与美化
开机界面：
背景图片
星星粒子动画
标题“羊羊大作战”左上角显示，白色填充 + 蓝色描边
开始按钮浮动动画
游戏主界面：
玩家球和敌人球均可显示圆形皮肤
食物、道具小球颜色随机
动态星空背景
左上角显示分数，血条，倒计时

⚙️ 编译与运行
安装 Qt（Qt 5.15 或以上版本推荐）
打开 Qt Creator，加载本项目 .pro 文件
确认 resources.qrc 中包含所有皮肤和背景图片
编译并运行项目
