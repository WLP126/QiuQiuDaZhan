🐑 羊羊大作战（QiuQiuDaZhan 改进版）

  基于 Qt 的趣味小游戏，灵感来源经典球球吞噬类游戏。
  
 玩家控制球球，通过吞噬小球和敌人球成长，同时躲避比自己大的球。

 🎮 游戏玩法

 | 功能       | 说明                             |
| -------- | ------------------------------ |
| 控制小球     | 鼠标拖动控制玩家球移动                    |
| 吞噬规则     | 吃掉比自己小的球或食物可增长，被比自己大的球碰撞时缩小并扣分 |
| 生命值 & 分数 | 分数降为 0 或被吞噬则游戏结束               |
| 敌人球      | 随机生成、持续出现，可吃小球成长，皮肤随机分配        |

🌟 界面与美化

开机界面

  ->背景图片 + 星星粒子动画

  ->标题 “羊羊大作战”

  ->开始按钮浮动动画

游戏主界面

  ->玩家球 & 敌人球圆形皮肤显示

  ->食物 & 道具颜色随机

  ->动态星空背景

  ->左上角显示分数、血条、倒计时

⚙️ 编译与运行

1.安装 Qt（推荐 5.15 或以上）

2.打开 Qt Creator，加载项目 .pro 文件

3.确认 resources.qrc 中包含所有皮肤和背景图片

4.编译并运行
